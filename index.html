Casos de Covid-19 no RS - Mapas Por Dias

Casos de COVID-19 confirmados nos municípios do Rio Grande do Sul. Os dados foram obtidos no site Base dos Dados ( ACESSE AQUI ). Os dados foram importados para o RStudio em formato .csv e tratados. As cartas temáticas categorizada e de densidade também foram geradas no RStudio com o uso de left join dos dados com o geom do pacote geobr(), para a plotagem foi usado o pacote ggplot2(). Na medida do possível, os colaboradores tentarão atualizar as cartas diariamente.
